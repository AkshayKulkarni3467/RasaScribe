

## Installation
1. Clone the repository:
    ```sh
    git clone https://github.com/AkshayKulkarni3467/RasaScribe.git
    ```
2. Navigate to the project directory:
    ```sh
    cd RasaScribe
    ```
3. Install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```
